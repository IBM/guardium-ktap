K-TAP Bundler Finder

Use this tool to identify the appropriate K-TAP bundler for your environment, available on IBM Fix Central. These installers are released outside the standard general release cycle and are built on a per-request basis in response to individual customer KTAP module requests.

  • Select your Guardium version
  • Select your operating system
  • Optionally, enter your kernel version (to find it, run uname -r in your Linux terminal)
  • Download the matching K-TAP module bundler from IBM Fix Central

After entering the OS kernel (uname -r), Guardium version, and operating system, the tool displays the kernel match type as either "EXACT" or "EXACT LIKE", if found.

  • EXACT — The exact kernel module is available.
  • EXACT LIKE — The exact module is unavailable, but a compatible module listed in ktap-combos.txt can be used and is treated the same as "EXACT", so no kernel request is required.

If neither match is found, use the tool to display all kernels available for the selected Guardium version and platform. Only kernels with "EXACT" matches from the latest KTAP bundle on Fix Central will be displayed.

If a lower version from the same kernel family is available (for example, X.Y.Z-A.1 for requested kernel X.Y.Z-A.5), you may enable Flex-Load to test compatibility by using --ktap_allow_module_combos for shell installation or KTAP_ALLOW_MODULE_COMBOS=Y through GIM.
